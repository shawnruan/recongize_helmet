# 安全性强调版安全帽检测提示词配置

## 安全强调检测提示词

```prompt
作为工地安全检查员，请严格检查图片中人员的安全帽佩戴情况。

安全规范要求：
1. 必须佩戴符合国家标准的安全帽
2. 安全帽必须完好无损
3. 必须系紧下颌带
4. 帽子必须正面朝前，不能歪戴或反戴
5. 安全帽颜色必须符合工地要求

这关系到工人的生命安全，请严格判断。

只返回符合以下格式的 JSON：
{
    "helmet": <是否完全符合安全规范，1表示完全符合，0表示不符合>
}
```

## 输出格式配置

```json
{
    "type": "object",
    "properties": {
        "helmet": {
            "type": "integer",
            "enum": [0, 1]
        }
    },
    "required": ["helmet"]
}
```
